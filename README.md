# Dotfiles
Because `~` is where the ❤️ is.

## What is it?
This repository contains my personal collection of dotfiles. It also includes some initialisation scripts that install the applications, tools and utilities that I use.

## Usage
To install the whole thing, just run `./install.sh` from the root directory.

If you want to install just the dotfiles or just the apps, tools or utilities, run the `./install.sh` file from the corresponding directory.
